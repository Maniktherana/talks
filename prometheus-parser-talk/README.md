# JSX Parser

A basic JSX parser and lexer implementation using Python and PLY (Python Lex-Yacc).

## Features

- JSX elements parsing
- JSX attributes support
- Nested JSX elements
- Text content
- Self-closing tags

## Quick Start

```bash
make install

make format

make test
```

### Cleanup

```bash
make clean
```

### Examples

```bash
make print-ast

make format-jsx
```
